# 用户研究标准作业流程（SOP）

覆盖问卷调研、用户访谈、行为数据分析、竞品研究四类场景的标准化流程。

## 流程概览

| 阶段 | 核心内容 |
|------|---------|
| 1. 需求承接 | 需求确认三要素 + 方法选择判断树 |
| 2. 研究设计 | 样本量规则、问卷规范、访谈提纲规范 |
| 3. 数据采集 | 问卷投放、访谈执行、行为数据获取 |
| 4. 数据分析 | 定量5步 + 定性4步，含质控要点 |
| 5. 报告交付 | 结构规范 + 交付前自查清单 |
| 6. 项目收尾 | 进度管理、归档沉淀、方法论复用 |

## 适用场景

- 问卷调研（定量）
- 用户访谈（定性）
- 行为数据分析
- 竞品研究
