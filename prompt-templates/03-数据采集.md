# 阶段三：数据采集 — 提示词模板

---

## 3.1 问卷投放检查清单（发给AI帮你自检）

```
我的问卷即将投放，请帮我检查投放前准备：

问卷平台：{问卷星/腾讯问卷/其他}
投放渠道：{公众号推文/社群/短信/APP弹窗/其他}
目标人群：{什么特征}
各分群配额：{各群目标样本量}
回收率预估：{参考历史数据或行业基准3%-5%}

请帮我：
1. 计算各分群的投放总量（目标样本÷回收率）
2. 生成投放前检查清单（渠道、定向、链接测试）
3. 设计每日监控指标（看什么数据、什么情况下补投）
4. 预判可能的欠采风险并给出应对方案
```

---

## 3.2 访谈记录整理（发给AI帮你结构化）

```
请帮我整理以下访谈记录，按三栏格式输出：

原始记录：
{粘贴访谈的原始记录或转录文本}

输出格式：
| 序号 | 原话引用（关键表述） | 行为归因（研究者分析） | 研究者判断 |

要求：
1. 原话引用保留用户原话，不改写
2. 行为归因是你对用户行为动机的分析
3. 研究者判断是你的核心结论（一句话）
4. 标注哪些发现是"核心发现"（高频+高影响）
5. 标注哪些发现需要和其他访谈交叉验证
```

---

## 3.3 数据口径核查（发给AI帮你验证）

```
我收到了以下数据，请帮我核查口径：

数据来源：{哪个系统/平台}
数据描述：{什么数据，什么时间范围}
去重逻辑：{按什么去重}
分母定义：{用什么做分母}

我的分析需求：{你要怎么用这些数据}

请帮我检查：
1. 数据口径是否和我的分析需求匹配
2. 去重逻辑是否一致（会不会出现同一个用户被算两次）
3. 分母定义是否统一（不同指标的分母是不是同一个东西）
4. 时间范围是否对齐
5. 如果有问题，我应该找数据侧确认什么
```
