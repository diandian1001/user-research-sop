# 阶段四：数据分析 — 提示词模板

---

## 4.1 定量数据清洗（发给AI帮你定规则）

```
我有一份问卷数据需要清洗，请帮我制定清洗规则：

数据来源：{问卷星/腾讯问卷导出}
总样本量：{多少份}
分群方式：{怎么分群}
问卷时长：{正常填写大概几分钟}

已知问题：{有没有已知的数据质量问题}

请帮我：
1. 制定无效样本剔除规则（答题时长、关键题空白、逻辑矛盾）
2. 每条剔除规则必须记录：规则依据、命中数量、对各分群的影响。阈值应根据预试或正式样本的完成时长分布、题目数量、逻辑矛盾、直线作答、注意力检查和开放题质量综合制定，不要使用孤立的固定时长阈值
3. 生成清洗报告模板（记录剔除数量、原因、占比）
4. 预估清洗后各分群的剩余样本量是否足以支撑分群分析目标（根据研究问题和统计需求判断，不使用固定门槛）
```

---

## 4.2 交叉分析（发给AI帮你找差异）

```
请帮我做以下交叉分析：

数据概况：{描述数据结构，比如"601份问卷，6类人群"}
核心变量：{你要对比的维度，比如"游戏类型偏好"}
分群变量：{按什么分群，比如"腾讯系/非腾讯系/不玩游戏"}

要求：
1. 列出每个分群在核心变量上的分布
2. 标注分群之间差异最大的维度
3. 对差异做统计检验（卡方/t检验），标注显著性；当检验假设不满足时（如期望频数不足），要求合并类别、使用适当方法或只做描述性报告
4. 根据检验方法、期望频数、样本结构、效应量和不确定性判断能否比较
5. 给出"所以呢"的初步解读（这个差异说明什么）
```

---

## 4.3 定性编码（发给AI帮你归类）

```
请帮我对以下访谈数据做主题编码：

访谈数量：{多少场}
受访者类型：{什么用户}

原始数据：
{粘贴多场访谈的关键发现/原话引用}

要求：
1. 逐条提取核心行为动机（不批量归纳）
2. 跨样本归并共性主题，标注出现频次（X/N场提到），明确频次仅是信号
3. 找出反例（和主流发现矛盾的个案）
4. 按与决策问题的相关性、对关键任务的影响、证据强度、覆盖的用户状态和场景综合判断重要性，分为核心发现/次级发现/观察备注
5. 每个主题附原话引用或行为证据作为支撑
```

## 4.4 结论审查（发给审查模型验证）

```
请审查我以下分析结论：

{粘贴你的分析结论}

审查标准：
1. 每条结论是否有数据或证据支撑（有数据支撑的需说明样本和来源；定性结论需记录Claim ID、参与者编号、原话或行为证据、场景、支持证据、反例和局限）
2. 结论是否过度推断（样本量能不能支撑这个结论）
3. 是否存在反例未被提及
4. 关键数字是否可信（有没有和常识矛盾的数字）
5. 结论之间是否有逻辑矛盾
6. 是否遗漏了重要维度
7. 只有研究问题涉及分布或频率时，才要求报告人数或比例

对每条结论给出：✅ 可靠 / ⚠️ 需补充证据 / ❌ 需修正
```
